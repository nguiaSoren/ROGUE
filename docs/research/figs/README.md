# Paper figures — reproducibility manifest

*Local/WIP (gitignored). Frozen figure data + figures for the adaptive-orchestration paper. Everything here regenerates from two commands; the `data/` snapshot is the citable source of every number in the paper.*

## Regenerate

```bash
uv run python scripts/export_paper_data.py   # 1. live Neon → docs/figs/data/ (CSVs + metrics.json)   [read-only]
uv run python scripts/paper_figs.py          # 2. docs/figs/data/ → docs/figs/*.png                   [offline, no DB]
```

Step 2 reads **only** the frozen `data/` snapshot, so figures are reproducible offline (proven: `env -u DATABASE_URL uv run python scripts/paper_figs.py` still works). Re-run step 1 only when the underlying runs change. `--only F2 F5` plots a subset.

## How the derived metrics are sourced (not hardcoded)

`metrics.json` is written by `export_paper_data.py`, which derives every value rather than asserting it:
- **graduations** are *queried*: `SELECT count(*) FROM attack_strategies WHERE first_breach_at BETWEEN <run start> AND <run end>`, where the window is `min/max(ladder_attempts.created_at)` for that run. (Validated: yields 0 / 3 / 4, matching the run analyses.)
- **escalation_spend** is parsed from each run's `done:` line in `/tmp/rogue_*.log`.
- **active/candidate snapshots** are derived from the current totals minus each later run's graduations (back-fill).
- **quota_sim** is re-run from `simulate_quota.py` (deterministic replay of the greedy run's logged rotation).

The `provenance` field in `metrics.json` records this inline.

## Honest-presentation notes (baked into the figures)
- **F4:** only N=2 measured points; the post-K=5 region is drawn dashed/shaded and labelled *hypothesized, not fitted*.
- **F5:** caption states win-share and breach-rate are different measures on different denominators (~1,800 balanced trials/model); the claim is the *rank inversion*, not a like-for-like ratio.
- **F7:** cells with N<20 trials are masked (greyed) — explicit about sparsity.
- **F6:** labelled a *simulation*; states it cannot predict whether a reached candidate breaches.

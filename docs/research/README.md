## Conventions

- **Figures:** `scripts/paper_figs.py` reads `docs/research/figs/data/` and writes PNGs to `docs/research/figs/`. Regenerate after any finding that changes the underlying numbers (CLAUDE.md "keep the research record current").
- **Tracked vs local:** drafts that are still moving or that duplicate internal planning are gitignored; writeups that are stable / partly public (grammar null-result, bandit, the judge paper) are tracked.
- **One topic per paper.** The scheduler/orchestration work and the judge-calibration work are *separate* papers (different subjects); the grammar null-result is its own study.
- **Provenance + honesty:** every reported number traces to `breach_results` / `data/calibration/` / a named external benchmark; mark pending/unmeasured results as pending (do not quote a number we haven't collected).
- [Oversight meaningfulness](oversight_meaningfulness.md) — Surface-2 human-gate: measuring a false-approve rate vs an independent key + the bias-laundering guard (novel; real number pending reviewers)
- [Skill-pool leakage](skill_pool_leakage.md) — Surface-3: measured adversarial leakage on a privacy-contained agent-skill pool (85% [70-100%]; canary ground truth; the first 10% was a rate-limit artifact; publishable)
- [Verified-promotion](skill_pool_promotion.md) — Surface-3: do accumulated skills help? 1/4 cleared the gate on held-out tasks (the gate works; reproduces SkillProbe); +3 bugs caught incl. a rate-limit artifact
- [Payload-embedding technique signal](payload_embedding_technique_signal.md) — Tier-A pilot: payload embeddings carry only a faint, non-separating technique signal (silhouette ≈ 0; n=45, MiniLM + prod embedder); confirms surface dominates technique → shelves the unified multimodal+multilingual embedding idea before Tier-B spend (negative result, on file)
- [Survival predictor (Q11)](survival_predictor.md) — black-box, embedding-free predictor of **cross-system-prompt survival**: ranks harvested attacks so a scan fires the ~14%-base-rate survivors first (offline back-test AUC 0.77, budget-saved 0.39 on 1,939 real pairs; live A/B gated). The per-*attack* black-box axis is the gap Kirch/Ball (white-box) and Helm (per-config) leave open.
- [Calibrated pre-fire evaluation gating (Q7)](prefire_scorer.md) — a **calibrated pre-fire skip policy**: decide *before firing* whether to spend target+judge calls on an (attack × target) pair, and **quantify the recall cost of skipping** — the number Zhang leaves unmeasured. $0 back-test over 1,845 embedded pairs: skip **42%** of trials at 80% breach recall but only **~7% at a production-safe 95%**; that steep curve *is* the finding, not the estimator. The contribution is the decision discipline — Platt calibration, a drift-guard that never skips OOD families, a 15% validation canary, visible skip accounting, byte-identical-when-off across all 3 scan surfaces; a small semantic-affinity term adds only AUC 0.688→0.696 and is reported as one feature among many, not an oversold headline. Q11's skip-policy sibling (Q11 orders, Q7 skips). Grounds on Kirch 2411.03343 + Zhang 2511.22044 + Galinkin 2412.01547.
- [SPRT early-stopping (Q6)](sprt_early_stopping.md) — treat the per-cell trial budget as **sequential**: Wald's SPRT (H0 p≤0.25 vs H1 p≥0.55) stops each cell once the breach verdict is statistically clear, recovering the same decision for fewer target+judge calls and reallocating trials to the borderline cells where a fixed `n=3` ASR was meaningless. Offline replay over 1,939 real cells: **22.1% calls saved at 99.8% agreement** (50–80% on high-trial cells); live % gated ~$35. Grounds on ConSol 2503.17587.
- [Cascade judge (Q2)](cascade_judge.md) — grade every trial with the **free regex heuristic first**, escalate to the paid LLM judge only when it isn't a confident non-breach: cheap tier is $0 (not a smaller paid model), so every short-circuit removes a full judge call for nothing, and an **asymmetric rail** (never assert a breach from the free tier) keeps the headline breach rate exactly what today's calibrated judge reports. $0 replay over 12,452 `breach_results`: **18.4% of judge calls saved at 99.8% agreement** at the safe default gate, **46.5% at a certified ≥95% floor** when the gate is lowered. Live % gated ~$35. Grounds on Ramírez 2405.02134 + Jung 2407.18370.
- [Noise-corrected calibration (Q4)](noise_corrected_calibration.md) — a **certification layer**: de-bias a judge-labelled breach rate against the judge's own **TPR̂/FPR̂** (estimated from the existing human-labelled 2×2) and emit a finite-sample **certified claim** ("true rate < α, Type-I ≤ ζ") — unblocking the previously-uncalibrated redaction/RTBF/user-safety/PII/memory judges. $0 replay over `breach_results`: raw **13.5% → corrected 1.0% [0–6.6%]** (most observed positives attributable to the judge's own FP floor); certifies < 20% at ζ=0.05. Novelty is the *integration* (validity guarantee + live benchmark + heterogeneous judges + production reporting), not the estimator (Rogan–Gladen/Feng/Lee). Per-judge headline gated on a **population-matched** calibration pass — which doubles as the experiment settling cross-population transfer. Grounds on Feng 2601.20913 + Lee 2511.21140.
